# Instructions

1. **Open your terminal** 
2. **Go to the Dashboard Directory** 
3. **Install the requirements.txt file** 
   ```bash
   pip3 install -r requirements.txt
   ```
4. **Run the app**
	```bash
	python3 app.py
	```
5. **You will be given the local host of the dashboard go to that url**
